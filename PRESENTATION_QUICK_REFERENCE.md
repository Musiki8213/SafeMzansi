## ‚ùì ANTICIPATED QUESTIONS & ANSWERS

**Q: Why Capacitor instead of React Native?**
A: "Capacitor allows code reuse from our React web app, reducing development time. We can maintain one codebase for web and mobile."

**Q: Why Vercel for deployment?**
A: "Vercel provides excellent serverless functions, automatic scaling, CDN, and easy deployment. Perfect for our Express backend."

**Q: How does the hotspot detection work?**
A: "I implemented a clustering algorithm that groups nearby reports, calculates density, and assigns color codes based on severity levels."

**Q: What was the biggest challenge?**
A: "Converting the Express app to work with Vercel's serverless architecture required restructuring the code to export properly as serverless functions."

**Q: How did you handle authentication?**
A: "JWT tokens for stateless authentication, works well with serverless architecture. Tokens stored in localStorage on client side."
