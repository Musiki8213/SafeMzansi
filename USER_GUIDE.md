## Frequently Asked Questions

### Q: Why don't I see my own reports on the Alerts page?
**A**: The Alerts page only shows reports from other users. Your reports appear on your Profile page.

### Q: Why is the safe route longer than the original?
**A**: The safe route takes a longer path to avoid all crime hotspots. Safety is more important than speed!

### Q: How often are notifications updated?
**A**: The app checks for new notifications every 3 seconds, so you'll see them almost instantly.

### Q: Can I delete someone else's report?
**A**: No, you can only delete reports that you created. This keeps the system fair and accurate.

### Q: What if I can't log in on mobile?
**A**: Make sure you've configured the API URL correctly. The configuration screen should appear when you first open the app.

### Q: How do I know if an area is safe?
**A**: Check the map colors:
- Brown = Generally safe
- Orange = Be cautious
- Red = Avoid if possible

## Need Help?

### Common Issues

**"Server Error" when logging in:**
- Check your internet connection
- Verify the API URL is configured correctly (mobile app)
- Make sure the backend server is running

**Map not loading:**
- Check your internet connection
- Refresh the page
- Clear browser cache and try again

**Notifications not appearing:**
- Make sure you're logged in
- Check that notifications are enabled in your browser/device settings
- Refresh the app

**Can't submit a report:**
- Make sure you're logged in
- Fill in all required fields (type, description, location)
- Check your internet connection
